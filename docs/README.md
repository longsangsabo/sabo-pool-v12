## 🚀 Quick Start

### For Developers
```bash
# 1. Read Developer Guide first
open docs/DEVELOPER_GUIDE.md

# 2. Setup development environment
git clone https://github.com/longsangsabo/sabo-pool-v12.git
cd sabo-pool-v12
pnpm install
pnpm dev

# 3. Understand the architecture
open docs/TECHNICAL_ARCHITECTURE.md
```
